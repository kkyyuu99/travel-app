처음 사용하시는 분께 — 가입 안내

안녕하세요! 이 앱은 노션처럼 복잡하지 않고, 여행 중 폰으로 빠르게 기록할 수 있게 만들어진 도구예요. 천천히 따라하시면 됩니다.

1단계 — 앱 설치 (한 번만)
  1. 안드로이드 폰: 크롬 브라우저로 kkyyuu99.github.io/travel-app 접속
  2. 주소창 옆 ⋮ 메뉴 → "앱 설치" 또는 "홈 화면에 추가" 누르기
  3. 아이폰: 사파리에서 같은 주소 → 공유 버튼 → "홈 화면에 추가"
→ 홈화면에 앱 아이콘 생김. 진짜 앱처럼 작동해요.
2단계 — 회원가입 (한 번만)
  1. 앱 첫 화면 위에 "⚪ 로컬 모드 · 로그인 →" 표시 누르기
  2. "회원가입" 탭 선택
  3. 이메일 + 비밀번호(6자 이상) + 이름(선택) 입력 → "가입 신청 & 인증코드 발송"
  4. 이메일 메일함 확인 (스팸함도 꼭 확인) → 6자리 숫자 코드 복사
  5. 앱으로 돌아와서 코드 붙여넣기 → "인증 완료 & 로그인"
3단계 — 자녀가 공유한 여행 받기
  1. 자녀에게 가입 완료 후 사용한 이메일 주소를 알려주세요
  2. 자녀가 그 이메일로 여행을 공유해주면, 자동으로 여행 목록에 나타나요
  3. 앱을 닫았다가 다시 열면 보일 수도 있어요 (자동 동기화)
여행 중 자주 쓰는 기능
  • 🗓️ 일정 — 일자별 동선이 시간순으로 정리됨. 가본 곳 ✓ 체크
  • 📝 메모 — 각 장소 카드 펼쳐서 ✏️ 누르고 자유롭게 기록. 사진도 첨부 가능
  • 💴 가계부 — 하단 [가계부] → 금액 입력 + 카테고리 선택 → 추가
  • 📍 구글지도 — 각 항목에 지도 미리보기. 탭 시 구글맵 앱 열림
